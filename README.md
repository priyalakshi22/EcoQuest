# 🌿 EcoQuest

> An interactive educational game that teaches teens about the environment through quizzes, levels, and real-world eco tips.

![EcoQuest Banner](./assets/banner.png)

---

## 🌍 About

EcoQuest is a web-based educational game designed for teens (ages 13–18) who want to learn about environmental issues in a fun, engaging way. Instead of passive reading, players **earn their knowledge** — progressing through levels, testing themselves with trivia, and walking away with tips they can actually use.

---

## ✨ Features

- 🎮 **Progression System** — Advance through levels as you learn, keeping motivation high
- 🧠 **Quizzes & Trivia** — Covers topics like climate change, biodiversity, pollution, and sustainability
- 🌱 **Real-World Eco Tips** — Each level unlocks practical actions players can take in everyday life
- 🌐 **Browser-Based** — No installation needed; works on any modern web browser

---

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Edge, Safari)

### Run Locally

```bash
git clone https://github.com/yourusername/ecoquest.git
cd ecoquest
# Open index.html in your browser, or use a local server:
npx serve .
```

## 🎯 How to Play

1. Open EcoQuest in your browser
2. Start at **Level 1** and answer environment-themed quiz questions
3. Complete each level to unlock the next and earn real-world eco tips
4. Challenge yourself to complete all levels!
